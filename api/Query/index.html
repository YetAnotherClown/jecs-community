Query

An object for filtering and iterating over entities with a specific set of Components.

iterating over Query​

</>
for  Id, ...any  in  Query  do

Iterating over a Query will return the Entity Id, followed by the values of your components, or nil if the component has no value.

cached​

</>
Query:cached() → Query

Returns a cached query.

CAUTION

A cached query is meant to be created once, and then used many times. Avoid creating a cached query in your actual systems, as to not create a cached query every frame. 

local function system(world: World)
	local enemies = world
		:query(Health, Position)
		:with(Enemy)
		:cached()

	return function()
		for id, health, position in enemies do
			-- ...
		end
	end
end
